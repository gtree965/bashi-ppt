巴适PPT v0.1.1 Windows 便携版
================================

一、启动

1. 请先完整解压 ZIP，不要在压缩包内部直接运行。
2. 双击 run_portable.bat。
3. 看到启动地址后，在浏览器打开：
   http://localhost:5100
4. 点击页面右上角齿轮，配置 AI 模型。

二、AI 模型

本地模型：
- LM Studio 默认地址：http://localhost:1234/v1
- Ollama 默认地址：http://localhost:11434/v1

云端模型：
- 阿里云百炼、硅基流动、OpenRouter 均可在设置界面配置。
- 其他 OpenAI 兼容服务请选择“自定义兼容接口”，填写服务地址、API Key 和模型 ID。
- 云端服务可能收费，并会接收你提交的材料。请先阅读服务商隐私政策。

三、隐私

- 本地模型模式下，文本生成请求发送到本机模型服务。
- 云端模型模式下，参考材料、提示词、大纲和讲稿请求会发送给云端服务商。
- Pixabay 图片搜索会把搜索关键词发送给 Pixabay。
- API Key 保存在本地 .env 文件，请勿分享或上传该文件。

四、常见问题

页面打不开：
- 确认命令窗口仍在运行。
- 检查 5100 端口是否被占用。
- 查看 launch_log.txt。

无法生成大纲：
- 检查模型服务是否启动。
- 检查模型 ID、API Key 和服务地址。
- 在设置界面点击“测试连接”。

PowerPoint 排版不同：
- 优先使用 Windows PowerPoint 或 WPS Office。
- 其他系统缺少相同字体时，可能发生换行变化。
- 导出后仍可在 PowerPoint/WPS 中编辑和替换字体。

五、停止

回到启动命令窗口，按 Ctrl+C。
