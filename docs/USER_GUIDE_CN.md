# 巴适PPT v0.1.1 使用指南

[English](USER_GUIDE.md) · [返回 README](../README_CN.md)

## 1. 选择 AI 来源

点击右上角齿轮进入设置。

### 本地模型

- LM Studio：先加载模型并启动 Local Server。
- Ollama：先安装并拉取模型。
- 点击“测试连接”，确认应用能够访问模型。

本地模型的优势是材料不需要发送给第三方；代价是速度和质量受电脑配置影响。

### 云端模型

设置界面可直接配置阿里云百炼、硅基流动、OpenRouter 和自定义 OpenAI 兼容接口。一般步骤是：

1. 在服务商控制台创建 API Key；
2. 在巴适PPT右上角齿轮中选择服务商；
3. 粘贴 API Key；
4. 选择推荐模型，或手动填写服务商页面上的 Model ID；
5. 点击“测试连接”，成功后保存。

阿里云百炼默认地址为 `https://dashscope.aliyuncs.com/compatible-mode/v1`，硅基流动默认地址为 `https://api.siliconflow.cn/v1`。通常不需要修改服务地址。

云端通常更快，但材料会离开本机，并可能产生费用。

## 2. 输入主题或材料

可以使用：

- 只有主题；
- 只有参考材料；
- 同时提供主题和参考材料。

同时输入时，参考材料决定主要内容边界，主题用于说明任务方向。

系统会根据主题范围或材料长度推荐页数。页数滑块会自动移动到推荐值，但你随时可以手动覆盖。

## 3. 选择生成模式

### 教学创作

允许模型补充一般性的教学背景、例子、提问和过渡。适合开放式备课。

### 严格依据材料

只整理、压缩和重组材料。流程如下：

1. 系统提取材料事实；
2. 你修改、删除或确认事实；
3. 系统生成带事实编号的大纲；
4. 你可以调整每页对应的事实；
5. 事实对应完整后才能生成严格材料讲稿。

事实编号审计只检查结构对应，仍需人工判断正文是否准确表达了事实。

## 4. 是否先生成备课文章

完整路径适合需要梳理内容的任务：

`主题/材料 → 备课文章 → 大纲 → 讲稿 → PPTX`

快速路径适合已有成熟讲义的任务：

`主题/材料 → 大纲 → 讲稿 → PPTX`

备课文章可以修改，并导出为 Markdown、DOCX 或 ODT。

## 5. 编辑大纲

生成后可以：

- 修改总标题；
- 修改每页标题和要点；
- 添加、删除或移动页面；
- 调整事实编号；
- 添加图示或搜索图片；
- 选择模板、主题和项目符号。

严格材料模式中新建的内容页不会自动继承事实，必须由用户选择依据。

## 6. 生成讲稿

讲稿设置包括：

- 讲课时长：5、10、20、30、45、60 分钟；
- 风格：课堂、主日学、家长沟通、正式表达。

讲稿按页生成，并写入 PowerPoint 的备注区。它更接近讲课提示卡，而不是强行填满整段授课时间的逐字稿。

## 7. 导出和课堂使用

导出的 PPTX 保留可编辑结构。建议：

1. 先在自己的 PowerPoint/WPS 中打开；
2. 检查教室电脑是否缺少字体；
3. 检查长标题、双语内容和图片；
4. 如果使用讲稿，提前确认演示者视图或备注区工作方式。

Keynote 和 LibreOffice 可能改变部分布局或备注处理方式。

## 8. 赞美诗歌词

赞美诗模式无需大模型。粘贴歌词后选择：

- 单语或双语；
- 每页行数；
- 简繁体转换；
- 投影主题；
- 是否添加标题页、阿们页。

预览分页后再导出。

## 9. 故障排查

### 模型连接失败

- 确认服务已运行；
- 检查地址、模型 ID 和 API Key；
- 检查 VPN、代理或防火墙；
- 云端服务请检查账户余额和限流。

### 生成很慢

- 本地模型可能超出硬件能力；
- 推理模型会输出大量思考 token；
- 可换用更小的本地模型或较快的云端模型。

### 图片搜索不可用

需要在设置中填写 Pixabay API Key。搜索关键词会发送给 Pixabay。

### 文字换行变化

不同系统和办公软件的字体度量不同。请在实际授课电脑上提前检查。
