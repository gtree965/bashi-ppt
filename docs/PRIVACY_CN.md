# 巴适PPT：隐私与数据去向

[English](PRIVACY.md) · [返回 README](../README_CN.md)

## 默认原则

巴适PPT本身不提供账户系统，不包含广告，也不内置使用情况分析或内容遥测。应用主要在本机 `127.0.0.1` 上运行。

## 本地模型

使用 LM Studio 或 Ollama 时，文本生成请求发送到本机配置的模型地址。巴适PPT不会主动把这些内容转发给其他 AI 服务。

请注意：

- 本地模型软件本身可能有自己的更新或联网行为；
- 用户配置的代理可能改变网络路径；
- 从网页搜索或远程 URL 插入图片仍会发生网络请求。

## 云端模型

使用 OpenRouter 或其他云端 OpenAI 兼容服务时，以下内容可能发送给服务商：

- 主题和参考材料；
- 事实抽取请求和已确认事实；
- 备课文章、大纲和修订请求；
- 逐页讲稿请求；
- 用于搜索图片的标题或关键词翻译请求。

云端服务的日志、训练使用、保留时间、地区限制和费用由服务商决定。巴适PPT无法替代用户与服务商之间的数据处理协议。

不要提交：

- 学生身份证号、家庭住址、联系方式；
- 未经授权的学生成绩、健康或辅导记录；
- 教会成员隐私、奉献或关怀记录；
- 商业秘密、考试题库、受保密协议约束的资料。

## Pixabay 图片搜索

启用图片搜索后，搜索关键词会发送给 Pixabay，图片文件会从 Pixabay 返回的 URL 下载。请遵守 Pixabay 的许可和服务条款，并核对图片是否适合教学用途。

## 本地保存

- API Key 保存在应用目录的 `.env` 文件中，为本地明文。
- 日志可能记录错误、模型名称、请求耗时和部分技术信息。
- 导出的 PPTX、文章和图片由用户选择保存位置。

发布错误日志前，请先删除 API Key、私人路径和敏感内容。

## 用户责任

选择云端服务意味着用户决定把内容发送给该服务商。用户应根据所在地区、学校制度和材料性质判断是否允许数据出境或第三方处理。

如果无法确认，请使用本地模型和不含个人信息的材料。
